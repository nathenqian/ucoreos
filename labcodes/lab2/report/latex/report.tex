\subsection{题目2}
一个被系统认为是符合规范的硬盘主引导扇区的特征是什么？
答：特征有如下几方面。
\begin{enumerate}
  \item 大小为512字节。
  \item 第511字节为0x55，第512字节为0xAA。
  \item 具体操作代码大小最大只有510字节。
\end{enumerate}
